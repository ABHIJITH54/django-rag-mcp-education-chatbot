# Django RAG + MCP Education Chatbot

This is a personal learning project for building an AI chatbot for an e-learning platform.

The chatbot can answer from uploaded PDF documents using RAG and also access live database data using Django ORM tools and MCP tools.

## Features

- Django + SQLite backend
- NVIDIA OpenAI-compatible API integration
- RAG over uploaded PDF documents
- Chroma vector database
- Live DB access using Django ORM
- MCP server for database tools
- Student profile lookup
- Course enrollment lookup
- Syllabus and pending topic tracking
- Exam date based study plan generation
- Mock test result analysis
- Attendance summary
- Live class details
- Payment status lookup
- Study material PDF download links
- Study plan PDF generation

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- ChromaDB
- MCP
- NVIDIA NIM API
- OpenAI Python SDK
- ReportLab
- PyPDF
